# Uber Trip Analysis

## Overview
This repository contains a Power BI dashboard that analyzes Uber trip data. The goal of this project is to explore trends related to trip volume, trip distance, fare distribution, time-based patterns, and overall rider behavior. The report provides a clear and structured view of how Uber trips vary across different time periods and locations.

## Features
- Interactive Power BI dashboard
- Analysis of trips by hour, day, and month
- Distance and fare distribution insights
- Pickup and drop-off pattern analysis
- Summary of key performance indicators
- Clean and simple data modeling using Power Query and DAX

## Tools and Technologies
- Power BI Desktop  
- Excel/CSV Dataset  
- Power Query for data cleaning  
- DAX for calculated measures
